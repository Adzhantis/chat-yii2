
Онлайн чат - одностраничное приложение, которое предоставляет возможность
пользователям оставлять сообщения в общем диалоговом окне.
Все данные введенные пользователями сохраняются в файле.
Оставлять сообщения могут только зарегистрированные пользователи,
регистрация производится без использования базы данных, записывать данные
рекомендуем в файл в формате JSON, необходимо оставить возможность
расширения приложения в будущем (подключение базы данных для регистрации и
сохранения сообщений).

Форма регистрации должна иметь следующие поля:
User Name (цифры и буквы латинского алфавита) – обязательное поле
E-mail (формат email) — обязательное поле
Homepage (формат url) – необязательное поле
Password

Для дизайна приложения используйте bootstrap.

К сообщению пользователь текстовый файл.
Текстовый файл не должен быть больше чем 100кб, формат TXT
Просмотр файлов должен быть в отдельном попап окне
